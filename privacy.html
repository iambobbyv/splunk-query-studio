Privacy Policy

Splunk Query Studio Souri Apps Last updated: February 2026
TL;DR — The short version
Splunk Query Studio collects zero data. No accounts. No analytics. No crash reporting. No network calls to external servers. Everything stays on your device.
1
Overview

This Privacy Policy describes how Souri Apps ("we", "us", "our") handles information in connection with Splunk Query Studio ("the App"), available on iOS, iPadOS, and macOS.

Splunk Query Studio is a fully offline SPL query builder. The App is designed to function entirely on your device and requires no account, login, or internet connection to operate.

2

We do not collect, transmit, or have access to any of the following:

Personal identifiers Usage analytics Crash reports Device identifiers Location data IP addresses SPL queries you build Saved presets Telemetry of any kind

3
Local Storage (On-Device Only)

The App stores the following data locally on your device to provide its core functionality. This data never leaves your device and is not accessible to Souri Apps:

Query History — your last 75 generated SPL queries, stored in device-local browser storage.

Saved Presets — named query presets you create, stored in device-local browser storage.

You can clear this data at any time by deleting the App from your device.

4
Network Activity

Splunk Query Studio makes no outbound network requests to any external server during normal use. All query generation is performed on-device using built-in logic and data.

Optional: Ollama AI (localhost only)

On macOS, the App can optionally connect to Ollama, a local AI runtime, if you have installed it separately. This connection is to localhost:11434 only — your own machine. No data is sent to Souri Apps or any external service. If Ollama is not installed, this feature is silently disabled.

When you tap an external link (such as the Souri Apps website), your default browser opens that URL. No personal data is transmitted as part of opening that link.

5
Third-Party SDKs & Services

Splunk Query Studio contains no third-party analytics, advertising, or tracking SDKs.

The App is built with Capacitor (an open-source iOS/macOS bridge by Ionic). Capacitor itself collects no data. No other third-party services are integrated.

6
Children's Privacy

Splunk Query Studio is a professional developer tool intended for adults. It is not directed at children under 13. Since we collect no data from any user, no special handling for children's data is required or applicable.

7
Changes to This Policy

If we ever change this policy, we will update the "Last updated" date at the top of this page. Any future version of the App that introduces data collection will present a new policy for your review before collection begins.

Given that the App currently collects no data, we do not anticipate material changes to this policy.
